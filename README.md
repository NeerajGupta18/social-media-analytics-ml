## 📌 Overview
This project performs **end-to-end analysis of social media platforms (YouTube + Facebook)** to extract insights and apply machine learning techniques for engagement optimization.

It combines:
- 📊 Data Analysis (EDA)
- 📈 Visualization
- 🤖 Machine Learning (Clustering)

---

## ⚙️ Workflow

### 🔹 Data Preprocessing
- Cleaned and handled missing values  
- Converted data types (dates, currency)  
- Feature engineering (time-based + engagement features)  

### 🔹 Exploratory Data Analysis
- Analyzed subscriber growth & video views  
- Category-wise and country-wise insights  
- Engagement trends across platforms  

### 🔹 Machine Learning
- Applied **K-Means Clustering**  
- Used **Elbow Method** for optimal clusters  
- Applied **PCA** for dimensionality reduction

- 🤖 Content can be segmented into engagement clusters
